Read `AGENTS.md` and all files under `docs/`.

Do not modify code yet.

Inspect the repository and report:

1. current backend and mobile architecture;
2. language and framework versions;
3. existing authentication approach;

5. build, lint, test, and run commands;

7. conflicts with the documented target architecture;
8. missing environment configuration;
9. security or tenant-isolation risks already present;
10. a minimal implementation sequence for Task 01.
