# Quỹ FC — Product and Codex Documentation

This package contains repository instructions and product specifications for a multi-tenant amateur football team fund-management MVP.

Start with:

1. `CODEX_USAGE.md`
2. `AGENTS.md`
3. `docs/IMPLEMENTATION_PLAN.md`
4. `codex-prompts/00_REPOSITORY_AUDIT.md`

The documents assume a default stack of React Native/Expo, Spring Boot 3, Java 21, PostgreSQL, JWT, Flyway, VietQR, and SePay. Codex must first inspect the actual repository and preserve compatible existing conventions.
